# Vicjoub

Turn a fresh Ubuntu installation into a fully-configured, beautiful, and modern web development system by running a single command. That's the one-line pitch for Vicjoub. No need to write bespoke configs for every essential tool just to get started or to be up on all the latest command-line tools. Vicjoub is an opinionated take on what Linux can be at its best.

Watch the introduction video and read more at [vicjoub.org](https://vicjoub.org).
